# アプリ概要

## テーマ候補

共通の趣味のマッチングアプリ

## 必要機能

- ログイン機能
- メッセージ機能
- プロフィール検索機能
- 通知機能
- ブロック機能
- チャット機能

## 必要画面

- ユーザー関連
  - ログイン画面
  - 新規登録画面
  - 編集画面
  - プロフィール画面
  - プロフィール編集画面
  - ユーザー一覧画面
- メッセージ関連
  - メッセージ画面
- 通知画面（ポップアップにするかも）

レスポンシブには対応しない。

## 仕様

### ログイン機能

ログイン画面からログインする。  
アカウントを所持していない場合、新規登録画面からアカウント作成を行う。

### いいね機能

ユーザー一覧画面から気になったユーザーにいいねを送る。  
いいねを送られたユーザーには通知が届き、いいねするとチャットができる。
